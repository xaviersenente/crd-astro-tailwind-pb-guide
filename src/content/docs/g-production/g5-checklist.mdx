## Responsive Design

### Points de rupture (breakpoints)

Tester sur :

- **Mobile** : 320px - 767px
- **Tablet** : 768px - 1023px
- **Desktop** : 1024px+
- **Large desktop** : 1440px+

### Checklist responsive

- ✅ **Navigation** : hamburger menu sur mobile
- ✅ **Images** : adaptatives et optimisées
- ✅ **Typographie** : tailles fluides
- ✅ **Grilles** : de 1 à 3-4 colonnes selon l'écran
- ✅ **Espacements** : ajustés par breakpoint
- ✅ **Formulaires** : inputs full-width sur mobile
- ✅ **Boutons** : taille minimale de 44x44px (touch target)
- ✅ **Tables** : scroll horizontal ou cards sur mobile
- ✅ **Modales** : plein écran sur mobile
- ✅ **Sidebar** : collapse sur mobile

## Accessibilité (A11y)

### Sémantique HTML

```astro
<!-- Bon -->
<header>
  <nav aria-label="Navigation principale">
    <a href="/">Accueil</a>
  </nav>
</header>

<main>
  <article>
    <h1>Titre principal</h1>
  </article>
</main>

<footer>
  <nav aria-label="Navigation du pied de page">
    <!-- ... -->
  </nav>
</footer>

<!-- Mauvais -->
<div class="header">
  <div class="nav">
    <span onclick="...">Accueil</span>
  </div>
</div>
```

### ARIA labels

```astro
<!-- Navigation -->
<nav aria-label="Navigation principale">
  <a href="/" aria-current="page">Accueil</a>
</nav>

<!-- Boutons -->
<button aria-label="Fermer le menu">
  <svg>...</svg>
</button>

<!-- Images décoratives -->
<img src="deco.jpg" alt="" role="presentation" />

<!-- Images informatives -->
<img src="event.jpg" alt="Concert de printemps au conservatoire" />

<!-- Form -->
<label for="email">Email</label>
<input id="email" type="email" required aria-required="true" />
```

### Focus visible

```css
/* global.css */
:focus-visible {
  outline: 2px solid #3b82f6;
  outline-offset: 2px;
}

button:focus-visible,
a:focus-visible {
  outline: 2px solid #3b82f6;
  outline-offset: 2px;
}
```

### Contraste des couleurs

Ratio minimum :

- **Normal text** : 4.5:1
- **Large text** : 3:1
- **UI components** : 3:1

Tester avec : [WebAIM Contrast Checker](https://webaim.org/resources/contrastchecker/)

### Prefers-reduced-motion

```css
@media (prefers-reduced-motion: reduce) {
  * {
    animation-duration: 0.01ms !important;
    animation-iteration-count: 1 !important;
    transition-duration: 0.01ms !important;
  }
}
```

## Cohérence UI

### Design tokens

Vérifier l'utilisation cohérente de :

```css
/* Couleurs */
--color-primary: #3b82f6;
--color-secondary: #8b5cf6;

/* Typographie */
--font-size-base: 1rem;
--font-size-lg: 1.125rem;

/* Espacements */
--spacing-4: 1rem;
--spacing-6: 1.5rem;

/* Bordures */
--radius-base: 0.5rem;
--radius-lg: 0.75rem;
```

### Composants uniformes

- ✅ Boutons : mêmes tailles et variantes partout
- ✅ Cards : structure identique
- ✅ Forms : styles cohérents
- ✅ Modales : même apparence
- ✅ Tooltips : position et style uniformes

### Espacement cohérent

```css
/* Utiliser une échelle d'espacement */
.mb-4  /* 1rem */
.mb-6  /* 1.5rem */
.mb-8  /* 2rem */
.mb-12 /* 3rem */
```

### Typographie cohérente

```astro
<h3 class="text-2xl md:text-3xl font-bold">Titre H3</h3>
<p class="text-base leading-relaxed">Paragraphe</p>
```
